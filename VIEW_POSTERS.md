## 💾 保存海报为图片

### 方法 1: 浏览器截图
1. 打开 HTML 文件
2. 按 Cmd+Shift+4 (macOS) 或 Print Screen (Windows)
3. 选择海报区域
4. 自动保存到桌面

### 方法 2: 浏览器开发者工具
1. 打开 HTML 文件
2. 按 F12 打开开发者工具
3. 右键点击海报 → 截图节点
4. 保存为 PNG

### 方法 3: 使用 html2canvas (在应用中)
- 前端已集成 html2canvas
- 用户点击"保存海报"按钮自动下载
