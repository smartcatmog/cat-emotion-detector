## 关键数据流

```
用户输入
  ↓
{
  mood_text: "今天很累",
  body_state: "身体不舒服",
  need: "休息"
}
  ↓
API 分类
  ↓
匹配到 "kun_kun_mao" (困困猫)
  ↓
查询 CATS["kun_kun_mao"]
  ↓
{
  name: "困困猫",
  explanation: "你不是懒...",
  suggestion: "今天不用完成...",
  avoid: [...],
  recovery_tags: [...],
  neighbor_cats: [...]
}
  ↓
查询 CAT_IMAGES["kun_kun_mao"]
  ↓
"/cats/困困猫.png"
  ↓
返回完整响应
  ↓
{
  catId: "kun_kun_mao",
  name: "困困猫",
  explanation: "...",
  suggestion: "...",
  notSuitable: [...],
  recoveryMethods: [...],
  neighbor: "纸箱猫",
  catPhoto: "/cats/困困猫.png" ← 关键！
}
  ↓
前端显示
  ↓
<img src="/cats/困困猫.png" /> ← 显示真猫照片！
```
