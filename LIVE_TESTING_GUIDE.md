## 🐛 故障排查

### 问题: 图片显示为插画而不是真猫照片

**检查步骤**:
1. 打开 F12 开发者工具
2. 进入 Network 标签
3. 生成猫签
4. 查看是否有图片请求
5. 检查请求状态

**可能原因**:
- Vercel 部署还未完成（等待 1-2 分钟）
- 浏览器缓存（清除缓存后重试）
- 网络连接问题

### 问题: 页面加载缓慢

**检查步骤**:
1. 打开 F12 开发者工具
2. 进入 Network 标签
3. 刷新页面
4. 查看加载时间

**可能原因**:
- 网络连接慢
- 服务器响应慢
- 图片文件过大

### 问题: 控制台有错误

**检查步骤**:
1. 打开 F12 开发者工具
2. 进入 Console 标签
3. 查看是否有红色错误信息

**可能原因**:
- API 返回错误
- 图片加载失败
- JavaScript 错误
