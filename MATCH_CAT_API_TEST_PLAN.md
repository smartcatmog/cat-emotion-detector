## System Prompt 核心特性

### 三步匹配逻辑
1. **第一步**: 原话关键词扫描 - 发现即匹配，跳过后续步骤
2. **第二步**: 原话无明确关键词时，用选项组合匹配
3. **第三步**: 以上都匹配到，用情绪强度兜底

### 关键词分类
- **社交回避类**: 装死猫、躲柜子猫
- **情绪委屈类**: 委屈猫、被遗忘猫
- **情绪烦躁类**: 炸锅猫、炸毛猫
- **焦虑等待类**: 等门猫、玻璃猫
- **身体疲惫类**: 因因猫、失眠猫
- **时间处境类**: 周日晚上猫、假期结束猫、考前猫、迷路猫
- **正面状态**: 撒欢猫、晒太阳猫、黏人猫
- **空白类**: 发呆猫、纸箱猫、假睡猫

### 绝对禁止规则
- ❌ 晒太阳猫禁止出现在任何负面情绪结输入中
- ❌ 撒欢猫禁止出现在任何负面情绪结输入中
- ❌ 黏人猫禁止出现在任何明显负面情绪结输入中
- ❌ 用户原话包含"委屈"时，禁止匹配躲柜子猫
- ❌ 用户原话包含"不想说话/不想和世界说话"时，禁止匹配晒太阳猫
- ❌ 用户原话包含任何情绪关键词，禁止匹配因因猫或晒太阳猫
